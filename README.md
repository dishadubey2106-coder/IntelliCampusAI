# IntelliCampus AI

A smart student productivity management system developed using Flask and MySQL.

## Features

* User Registration and Login
* Task Management System
* Productivity Analytics Dashboard
* Pie Chart Visualization using Matplotlib
* PDF Report Generation using ReportLab
* CSV Export Functionality
* Smart Attention Monitoring using OpenCV
* Face Detection Based Focus Score
* Session Management and Authentication

## Tech Stack

* Python
* Flask
* MySQL
* OpenCV
* Matplotlib
* ReportLab
* HTML
* CSS

## How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Create MySQL database:
   intelligentcampus

3. Run:
   python app.py

4. Open:
   http://127.0.0.1:5000
